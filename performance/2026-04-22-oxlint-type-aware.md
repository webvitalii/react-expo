# Performance: Oxlint + Type-aware

https://oxc.rs/docs/guide/usage/linter/type-aware

## Oxlint:

```bash
$ npm run lint

Found 0 warnings and 0 errors.
Finished in 19ms on 70 files with 92 rules using 8 threads.
```

## Oxlint Type-aware:

```bash
$ npm run lint -- --type-aware

Found 15 warnings and 0 errors.
Finished in 916ms on 70 files with 107 rules using 8 threads.
```
